# Wanderlist

A premium travel discovery application built with React, Node.js, and MongoDB.

## Prerequisites
*   Node.js
*   MongoDB (Must be running locally on default port 27017)

## Getting Started

1.  **Install Dependencies** (if you haven't already):
    ```bash
    cd server && npm install
    cd ../client && npm install
    ```

2.  **Run the Application**:
    Double click `run.bat` 
    OR
    Run manually:
    *   Terminal 1: `cd server && npm run dev`
    *   Terminal 2: `cd client && npm run dev`

3.  **Open Browser**:
    Go to `http://localhost:5173`

## Features
*   **Explore**: reliable country data via REST Countries API.
*   **Wanderlist**: Save your favorite destinations (Requires Account).
*   **Reviews**: Share experiences and rate countries (Requires Account).
*   **Auth**: Secure Login/Register with JWT.
